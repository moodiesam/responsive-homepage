PROJECTS

Workout Tracker App

The Workout Tracker App is a comprehensive web application designed to help users create and manage their exercise routines. This project showcases my skills in full stack development, utilizing the MERN stack (MongoDB, Express.js, React, and Node.js) to deliver a seamless and responsive user experience. Users can register, browse a curated list of exercises, build custom routines, and save these routines to their profiles.

Etch-a-Sketch

The Etch-a-Sketch Game is an interactive and artistic web application that emulates the classic drawing toy. This project highlights my proficiency in JavaScript, particularly in using event listeners to create dynamic and responsive user interactions. As users move their mouse over the grid of tiles, each tile changes color, allowing for creative and playful artwork to be made directly in the browser.

Memory Game

The Memory Card Game is an engaging and challenging web application built with React, where users aim to click each image only once, with the twist that all the cards shuffle after each click. This project demonstrates my ability to work with APIs to fetch images dynamically and implement complex game logic using JavaScript. Additionally, it showcases my skills in managing state in a React application to ensure smooth and responsive gameplay.

Inventory Tracker

The Inventory Tracker application is a robust web application designed to manage the inventory of a made-up store. Built with Express and using Pug as a templating engine, this project demonstrates my ability to create full stack applications that utilize CRUD operations to interact with a MongoDB database. Users can add, view, update, and delete items from the inventory, showcasing my skills in backend development, database management, and server-side rendering.

Battleship

The Battleship Game is a classic strategy game recreated with a focus on JavaScript fundamentals and test-driven development. This project highlights my proficiency in essential JavaScript concepts such as prototypes, closures, promises, and event loops. By implementing these concepts, I ensured the game logic is robust and maintainable. The development process was guided by rigorous testing to ensure a reliable and enjoyable user experience.

Weather App

The Weather App is a straightforward and user-friendly application that allows users to check the weather for any location they enter. By leveraging public APIs, this project demonstrates my ability to fetch and display real-time data. The app provides current weather conditions, showcasing my skills in API integration and frontend development.
